## 部署

+ 安装 scrapyd
```bash
pip install scrapyd
pip install scrapyd-client # 上传工具(scrapyd-client)
```

+ 发布
    1) 拷贝scrapyd-deploy到爬虫目录下

    2) 修改爬虫的scapy.cfg文件

        1、去掉url前的注释符号，这里url就是你的scrapyd服务器的网址； 

        2、deploy:127表示把爬虫发布到名为127的爬虫服务器上，deploy:后的名字可以自己定义；
    
    3) 上传scrapy到scrapyd




## 算法：
深度优先算法-> 递归

广度优先算法-> 队列
